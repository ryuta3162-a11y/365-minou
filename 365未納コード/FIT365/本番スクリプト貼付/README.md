# 本番スクリプト貼付（比較・調査用）

スプレッドシートの Apps Script エディタからコピーした **本番のコード** を置く場所です。  
ここに置いたファイルは **実行用ではありません**（リポジトリの正本との差分確認用）。

## 正本（開発・デプロイ用）

| 用途 | パス |
| --- | --- |
| FIT365 各店 | `../Code.gs` と `../dialog.html` |
| FIT365 全店 | `../FIT365_全店実施管理ブック_GAS/Code.gs` |

## 店舗フォルダ

| 店舗 | 貼り付け先 |
| --- | --- |
| 志木店（サンプル） | `志木店/Code.gs`・`志木店/dialog.html` |

全店ブック用を貼る場合は `全店ブック/Code.gs` を用意予定（必要になったら作成）。

## 貼り付け手順（志木店）

1. 志木店の **各店未納ブック** を開く → **拡張機能 → Apps Script**
2. 左の `Code.gs` を開き、**エディタ内の全文**をコピー → `志木店/Code.gs` に**上書き貼り付け**
3. 左の `dialog.html`（無い場合はファイル追加してから）を同様に → `志木店/dialog.html` に貼り付け
4. **プロジェクトの設定**でスクリプトのタイムゾーン・トリガー一覧をメモしておくと調査に便利（コードファイルには含まれません）

貼り付け後、チャットで「志木店を貼った」と伝えてもらえれば、正本 `../Code.gs` との差分を確認できます。
