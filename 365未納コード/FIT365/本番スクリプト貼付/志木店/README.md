# 志木店 — 本番スクリプト貼付

志木店の **各店未納スプレッドシート** に紐づく Apps Script から、次の2ファイルに貼り付けてください。

| ファイル | Apps Script 側の名前 | 貼り付け先 |
| --- | --- | --- |
| `Code.gs` | `Code.gs`（既定の .gs） | このフォルダの **`Code.gs`** |
| `dialog.html` | `dialog.html` | このフォルダの **`dialog.html`** |

- ファイル内を**すべて選択して削除**し、エディタの内容を**そのまま全文**貼り付けて問題ありません。
- 貼り付け前のプレースホルダコメントは消えて構いません。

※ 全店「実施管理ブック」用の GAS は別プロジェクトです。志木店フォルダには **各店ブック用だけ** を入れてください。
