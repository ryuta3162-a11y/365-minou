| フォルダ        | 用途                      | 主なファイル                   |
| ----------- | ----------------------- | ------------------------ |
| **FIT365/** | FIT365 未納・請求まわり         | `Code.gs`, `dialog.html` |
| **JOYFIT/** | JOYFIT（ジョイフィット）未納・請求まわり | `Code.gs`, `dialog.html` |


ルート直下に `Code.gs` は置かない構成にしています（どちらのプロジェクトか迷わないため）。
